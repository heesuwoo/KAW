1.  컴포넌트 분리
	- 메인/마이페이지/관리자페이지
		1) 메인 컴포넌트 - 검색 기능, 예약 날짜 선택, 지도
		2) 마이페페이지 컴포넌트 - **카드 형식 목록 조회 - 마이페이지 - 스크랩좋아요 - 디지털 콘텐츠/이용자 갤러리
		3) 관리자 컴포넌트 - 간단한 조회/수정/입력 폼, 검색 기능

2. 공통 컴포넌트 분리 
	- 푸터는 다 공통, 헤더는 메인+마이페이지(공통 헤더(상위 메뉴 -> 메인에서는 메뉴가 안보이게 분기처리 필요))공통/ 관리자 페이지에서는 다른 헤더
	- **테이블 그리드 조회 -> 오픈 api 사용(https://m.blog.naver.com/deersoul6662/222053765452)
	- 조회 슬라이더(메인, 관리자페이지)
	- 이용자 갤러리(메인- 수정버튼 안보이게/ 작성자 이메일 보이게, 마이페이지- 수정버튼 보이게 및 작성자 이메일 안보이게)
